# Array Method: find()

## Overview
The `find()` method returns the **first element** in the array that satisfies the testing function.

## Key Concepts
- Returns the **element itself** (not an array)
- Returns `undefined` if no element matches
- Stops searching once a match is found (efficient)
- Does **not mutate** the original array
