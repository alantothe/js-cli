# Array Method: splice()

## Overview
The `splice()` method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

## Key Concepts
- **Mutates** the original array (unlike slice)
- Takes parameters: `start`, `deleteCount`, and optional elements to insert
- Returns an array of deleted elements
- Can be used to remove, insert, or replace elements
