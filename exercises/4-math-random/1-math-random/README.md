# Math & Random

## Overview
The Math object provides mathematical functions and constants for calculations.

## Key Concepts
- Math methods: `abs`, `round`, `floor`, `ceil`, `max`, `min`, `pow`, `sqrt`
- `Math.random()` generates numbers between 0 and 1
- Math constants: `Math.PI`, `Math.E`
- Useful for games, animations, and calculations
