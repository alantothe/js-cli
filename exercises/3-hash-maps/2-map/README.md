# Map (ES6 Hash Map)

## Overview
Map is an ES6 data structure specifically designed for key-value pairs.

## Key Concepts
- Keys can be any type (objects, functions, primitives)
- Better performance for frequent additions/deletions
- Has useful methods: `set`, `get`, `has`, `delete`, `clear`
- Maintains insertion order
