# Set (ES6 Unique Values)

## Overview
Set is an ES6 data structure for storing unique values.

## Key Concepts
- Only stores unique values (duplicates are automatically removed)
- Any type can be stored
- Methods: `add`, `has`, `delete`, `clear`
- No key-value pairs, just values
- Great for removing duplicates
