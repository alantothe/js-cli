# String Methods

## Overview
Strings have many built-in methods for manipulation and analysis.

## Key Concepts
- Methods: `length`, `charAt`, `toUpperCase`, `toLowerCase`, `indexOf`, `includes`, `slice`, `split`, `trim`, `replace`
- Strings are immutable - methods return new strings
- Chain methods together for complex operations
- Common for text processing and validation
