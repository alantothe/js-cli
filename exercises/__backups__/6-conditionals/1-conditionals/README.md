# Conditionals (if/else & switch)

## Overview
Conditionals control program flow based on conditions.

## Key Concepts
- `if` and `if-else` statements for binary decisions
- `if-else-if` chains for multiple conditions
- `switch` statements for matching specific values
- Ternary operator (`? :`) for inline conditions
- Logical operators: `&&`, `||`, `!`
